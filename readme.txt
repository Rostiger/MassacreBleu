 QUADCORE  is a local multi-player game for up to four players made with Processing.
 The game supports PS3, XBox and NES USB controllers. Alternatively, if your keyboard has a number block, you can (uncomfortably) play with the keyboard:


 Player 1
 -------------------------------------------
 Move: 			Up, Down, Left, Right Arrows
 Shoot: 		/
 Use Item: 		Shift
 Start: 		Space

 Player 2:
 -------------------------------------------
 Move: 			W, A ,S ,D
 Shoot: 		F
 Use Item: 		R
 Start: 		Q

 Player 3:
 -------------------------------------------
 Move: 			I, J, K, L
 Shoot: 		H
 Use Item: 		Y
 Start: 		O

 Player 4:
 -------------------------------------------
 Move: 			8, 5, 4, 6
 Shoot: 		0
 Use Item: 		1
 Start: 		2
